# B2B IoT CRM Pipeline Analysis

End-to-end sales funnel analysis for a B2B smart solutions business.
Identifies where qualified deals die, why, and quantifies the impact
of a pricing intervention using a controlled before/after comparison.

## Tools

Python (Pandas) · PostgreSQL · Power BI · PowerPoint

## Key findings

- 79% of qualified deals die at Proposal stage
- Pricing and competitor presence account for 76% of competitive deal losses
- Fast-track pricing approval improved competitive win rate from 2.8% to 21.4%
- Deals stalling at Proposal beyond 14 days have near-zero win rate
